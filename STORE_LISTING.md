## Short Description (under 100 characters)

## Full Description
Lexer is an AI-powered code editor built for Windows developers
who want full control over their AI tools.

Key features:
- Monaco Editor (same engine as VS Code)
- AI chat with full codebase context — ask questions about your
  entire project, not just the open file
- Ghost text inline completions as you type
- BYOK — bring your own API key for NVIDIA NIM, Groq, or any
  OpenAI-compatible provider
- Local AI support via Ollama — 100% private, no data leaves
  your machine
- Workspace indexing with vector embeddings for codebase-aware AI
- Integrated terminal panel
- Multi-tab editor with auto-save
- No subscription, no login, no cloud required

from your machine to your chosen provider using your own API keys.
